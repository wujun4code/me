1. .NET Core 后端/业务需求，我很有兴趣一起创造
2. 2C 端的移动应用
3. 2B 端的企业级 SaaS
4. Angular 的前端 Web App
5. WPF 的好看好用的客户端，并且 C 端客户足够多或者潜力足够大
